# 概述

## 简介

`vue-scaff` 是一个基于 Aspect 理念而生的企业级 Vue 脚手架，旨在帮助开发者快速构建 vue 应用程序。它能构建的应用类型包括但不限于 `Web`、`H5`、`小程序（Uni-App）`、`桌面应用（Electron）` 等。

## 环境（node 与 npm 版本）

:::tip 温馨提示

- node.js >= 16+
- npm >= 7.10+

:::
