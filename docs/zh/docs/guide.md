# 入门

## 概述

`vue-scaff` 是一个独具匠心的脚手架，可以帮助您快速构建 vue 应用程序。 它可以构建的类型包括但不限于 `Web`, `H5`, `Desktop (Electron)`，等。

## node.js 和 npm 的版本

:::tip 温馨提示

- node.js >= 16+
- npm >= 7.10+

:::
